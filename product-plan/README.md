# Calvo Creativo OS - Product Plan

This is the exported product plan for **Guitar Curves**. It contains everything an AI agent or developer needs to build the application.

## Contents

- **prompts/**: Ready-to-use prompts to start coding.
  - `one-shot-prompt.md`: The master prompt for the entire build.
- **data-model/**: The entity definitions (Products, Collections, etc.).
- **design-system/**: Color palette (Golden Sculpt) and Typography.
- **shell/**: Specification for Header, Footer, and Mobile UX.

## How to use

1. Open your AI coding assistant (Claude, GitHub Copilot Workspace, Cursor).
2. Paste the content of `prompts/one-shot-prompt.md`.
3. Provide the referenced files (`data-model.md`, `colors.json`, `spec.md`) as context.
4. Watch it build!
